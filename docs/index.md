# 给 Vue 工程师的 Java 全栈教程

> 你已经会写 Vue，现在要把触角伸向后端。这份教程专为**前端工程师**设计——
> 用你熟悉的 JS/TS/Vue 概念做桥梁，带你从 Java 语法一路走到能独立写出 **Spring Boot 后端**，
> 并完成 **Vue + Spring 全栈打通**。

---

## 这份教程适合谁

- ✅ JS/TS 熟练，懂 HTTP / REST / JSON / 异步
- ✅ 但**从没碰过**静态类型、JVM、构建工具、后端
- ❌ **不适合**：想系统啃 JUC 源码 / JVM 调优的资深 Java 工程师（这是「能写后端」的实战向，不是八股大全）

## 你将学会什么

读完你能：独立搭建 Spring Boot 项目 → 设计数据库 → 写 REST 接口 → 实现 JWT 登录认证 →
用 Vue 前端对接自己的后端 → 联调、踩坑、部署上线。

## 技术栈一览

| 维度 | 选型 | 为什么 |
|---|---|---|
| Java | **JDK 8 风格** + 《从 8 到 17》新特性章 | 国内公司 / 面试最常见；新特性章补现代语法 |
| 构建工具 | **Maven** | 心智模型最接近你熟悉的 `package.json` |
| 后端框架 | **Spring Boot 2.7.x** | 支持 Java 8，国内主流；新特性章带 3.x 迁移 |
| 数据库 | **MySQL** | 国内绝对主流 |
| 持久层 | **MyBatis-Plus** | SQL 透明可控，对前端人最直观 |
| 前端（全栈章） | **Vue 3 + Vite + Pinia + Axios** | 你已经会的栈 |
| 文档 | **mkdocs + Material** | 本教程本身就用它构建 |

!!! info "关于本机 JDK 版本"
    本书代码以 **Java 8 风格**为基线（Maven 用 `--release 8` 交叉编译）。
    你本机若是 **JDK 17**，完全能正常构建运行这些代码；如果你公司还在 **JDK 8**，这些代码也能直接跑。
    《第五篇》会专门讲从 JDK 8 到 17 的演进。

## 怎么读

1. **第一篇**先看《Vue/JS 工程师的 Java 速查表》——这是全书类比桥梁，建立"JS→Java"的映射直觉。
2. **第二篇**语言基础：每章都带"Vue 里相当于什么"，快节奏扫盲。
3. **第三篇** Spring Boot：进入实战，跟着敲一个真实的后端。
4. **第四篇**全栈：把 Vue 前端接上你的后端，跑通闭环。
5. **第五篇**进阶：JDK 17 新特性 + Spring Boot 3.x + JVM 入门。

## 这份教程怎么保证代码可信

书里**每一个代码示例都是真实可编译的**，存放在仓库的 `code/` 目录（Maven 工程）。
文档页面通过 snippet 引用**直接嵌入**这些 `.java` 文件——**你看到的就是编译的那份**，
改一处、自动同步，零漂移。想验证：在仓库根目录跑 `mvn -f code/pom.xml compile`。
